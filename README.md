# Jason Bandy Portfolio Dashboard

A static portfolio dashboard for Jason Bandy, focused on platform engineering, AI/ML systems, enterprise mobile infrastructure, and engineering leadership.

## Live Site

Netlify preview: https://app.netlify.com/drop/lustrous-cendol-79d229

## Contents

- `index.html` - portfolio dashboard markup
- `styles.css` - responsive dashboard styling
- `script.js` - project filter interaction
- `assets/operations-dashboard-hero.png` - generated hero visual
- `assets/jason-bandy-resume.pdf` - downloadable resume
- `blog/` - software leadership and mobile architecture articles

## Content Notes

The work examples are intentionally anonymized and generalized. They are written to demonstrate architecture judgment, leadership approach, and delivery patterns without exposing company-confidential data, internal URLs, account IDs, proprietary service names, or operationally sensitive details.

## Run Locally

Open `index.html` directly in a browser.

## Deploy

This is a plain static site and can be deployed to Netlify, Vercel, Cloudflare Pages, or GitHub Pages.
